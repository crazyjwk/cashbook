1. 회원가입
- png, jpg, jpeg 사진 업로드해서 회원가입 가능

2. 회원정보 조회
1) admin 로그인시
- 현재 회원가입된 회원들의 목록을 볼 수 있음
- 회원들을 강제탈퇴 시킬 수 있음(이때 가계부와 수입/지출 입력도 같이 지워짐 게시판과 댓글은 일부러 지우지 않음)

2) 일반회원으로 로그인시
- 회원정보 수정은 사진을 업로드 해야만 수정이 됨

3. 가계부 관리
- 가계부 리스트 페이징 구현
- 가계부를 각각 따로 생성해서 관리 가능
- 일별, 월별, 월별비교 자유롭게 이동 가능
- 날짜별로 금액 전부 출력
- 수입/지출 입력 비동기 처리 구현
- 수입/지출 수정, 삭제 구현

4. 게시판
- 게시판 작성, 수정, 삭제 가능 (해당 memberId만 가능)
- admin은 일반회원의 게시판, 댓글 삭제 가능
- admin의 댓글은 별도의 색깔로 구분하게 함
- 댓글 작성, 삭제 가능
- 게시판, 댓글 페이징 구현
